## Scope & Constraints

Across all artifacts, the following principles apply:

- Work was completed using **provided or mandated datasets and requirements**
- No proprietary, confidential, or internal operational data is included
- Projects prioritize **accuracy, defensibility, and operational realism**
- Tooling choices reflect what is commonly available in enterprise environments

This mirrors how most real-world analytics, reporting, and project work is actually performed.
